# PHYSIS 🌿
**Smart Homeostasis & Personal Development Agent**

Physis is a cutting-edge web application designed to bridge the gap between AI-driven analysis and personal well-being. By utilizing the Google Gemini 1.5 Flash model, Physis analyzes user intent and goals to provide a personalized dashboard focused on achieving homeostasis through specific biological and psychological "niches."

## 🚀 Key Features
- **AI-Powered Onboarding:** Real-time goal analysis using Generative AI (Gemini) to categorize users into niches: *Serenity*, *Prosper*, or *Scholar*.
- **Internationalization (i18n):** Native support for English and Spanish, detecting browser language and allowing seamless manual switching.
- **Full-Stack Architecture:** 
  - **Frontend:** React (Vite) + Tailwind CSS + Framer Motion for a fluid, premium UX.
  - **Backend:** Node.js + Express with PostgreSQL (Sequelize ORM).
- **Responsive Dashboard:** Adaptive interface that reflects user status, current protocols, and AI-generated biological rhythms.

## 🛠️ Technical Stack
- **Frontend:** React.js, Axios, i18next, Lucide-React, Framer Motion.
- **Backend:** Node.js, Express, Sequelize, CORS, Dotenv.
- **AI Service:** Google Generative AI (Gemini 1.5 Flash API).
- **Database:** PostgreSQL.
